Inkfish CLI

Inkfish CLI was a tool I wrote during the Spring 2021 semester to grade assignments from inside of my editor (instead of using a web UI) for Inkfish, a Northeastern grading website. It can download assignments to be graded, and can scan the code for grading comments to submit to inkfish, using requests and webscraping.

A short demo showing how the CLI works, cut and cropped for student confidentiality.
